RealTime Sign Detection and Recognition

The main aim of our project was the detection and recognition of sign language actions. We used the Google Mediapipe library to collect the key points of our hands, which we captured using the Python OpenCV library and stored them in an array. Using the key points, we trained the model using the LSTM layers. We also tested the model using the live feed of the camera. At last, We created a Web Application using the Flask framework in which we created a user login system by database connectivity.
